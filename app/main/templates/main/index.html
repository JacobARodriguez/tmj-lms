TMJ — Track My Journey

TMJ helps students understand their learning progress with clarity. Each course provides transparent progress bars, module completion indicators, badges, streak updates, and personalized reminders—helping learners stay organized, motivated, and on track.

  • Complete MVP: Secure login/logout, multi-course dashboard, detailed course view, completion badges, per-module notes, streak tracking, and automated reminder messages.
  • Real demo data: Includes a pre-seeded user with multiple courses, modules, progress records, and example notes for an instant walkthrough.
  • Built for expansion: Future enhancements can extend TMJ with instructor dashboards, additional achievement badges, and advanced module workflows.
